## Sort Options
| Field | Description |
|-------|-------------|
| `Recency` | Most recent first |
| `Quality` | Highest quality score first |
| `Rarity` | Highest rarity score first |

Direction: `Desc` (descending) or `Asc` (ascending)

## Date Filters
- `30` - Last 30 days
- `60` - Last 60 days
- `90` - Last 90 days
- `alltime` - All time (default)
- `after` - After specific date (use with `reportedAfter`)

## Score Filters
- `qualityScore`: 0-5 (minimum quality)
- `rarityScore`: 0-5 (minimum rarity)
